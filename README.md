# CrypTrack

A responsive React application for real-time cryptocurrency data display, featuring API integration, advanced search, sorting options, and customizable charts using Chart.js for enhanced data exploration.

## Features

- Live previews of cryptocurrency prices
- Search bar to search for specific cryptocurrency
- Prices can be viewed in 8 different currencies
- Market trends of the crypto can be checked for different time periods along with charts.


## Tech Stack

**Client:** JavaScript, React, Chart.js, BootstrapCSS

**Server:** Firebase, Node.js

**APIs & Data:** CoinGecko
